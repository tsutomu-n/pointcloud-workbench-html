# Security Policy

## 基本方針

PointCloudWorkbenchは、ユーザーが選択したLAS/LAZファイルをサーバーへアップロードしません。
ファイル処理はブラウザー内で行います。

## 報告対象

以下を見つけた場合はIssueまたはメンテナへ連絡してください。

- 点群ファイルが外部送信される可能性
- 任意コード実行の可能性
- XSS
- 不正な外部通信
- 依存ライブラリの脆弱性
- 地図タイル利用時の情報漏えいリスク

## 非対象

以下は仕様上の制約として扱います。

- 低性能PCでの処理遅延
- 巨大LAS/LAZの読み込み失敗
- Safari/Firefox非対応
- 外部地図タイルのCORS制約

## セキュリティ設計

- 点群ファイルはローカル処理
- サーバーアップロードなし
- 自動telemetryなし
- telemetry endpointなし
- error report自動送信なし
- 診断情報はユーザーが手動コピーした場合だけIssue等へ貼り付ける
- 手動診断情報にはLAS/LAZファイル本体とファイル名を含めない
- Cloudflare / Pagesは静的配信中心
