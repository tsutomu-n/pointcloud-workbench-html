PointCloudWorkbench

LAS / LAZ をブラウザだけで確認できる単一HTML点群ワークベンチです。 GitHub Pages では概要確認と軽い体験導線を提供し、実アプリ本体は既存の PointCloudWorkbench.html をそのまま配信しています。

Open Demo Download Sample LAS View Repository

Quick Facts

Single File ビルド不要の HTML 配布
LAS / LAZ ローカル読込とサンプル導線
GitHub Pages そのままURL共有できるデモ
Bundled Sample 小規模の合成 LAS を同梱

実アプリ内では サンプルデータを使用 から同梱 LAS と合成サンプルを選べます。 自分の `.las` / `.laz` をドラッグ&ドロップして使うこともできます。

How To Try

  1. Open Demo でアプリ本体を開く
  2. サンプルデータを使用 から Pages 用サンプルを選ぶ
  3. 品質を選んで 3D / 2D 表示や統計を確認する

Browser Notes

  • Chrome / Edge / Firefox の最新版を想定
  • WebGL と File API が必要
  • 実行時に `three.js` と `laz-perf` を CDN から読み込む

Demo Constraints

  • 同梱サンプルは軽量な合成 LAS です
  • LAZ はアプリ本体内でローカル読込の確認ができます
  • ネットワーク未接続では初期化に失敗する場合があります
