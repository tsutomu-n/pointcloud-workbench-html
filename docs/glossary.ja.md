# 用語集

## LAS

点群データの標準的なファイル形式です。

## LAZ

LASを圧縮した形式です。読み込み時に展開処理が必要です。

## 点群

3D空間上の多数の点の集合です。地形、道路、法面、構造物などを表します。

## Preflight

読み込み前の事前確認です。

## 表示率

元点数に対して、実際に表示する点数の割合です。

## CRS診断

LAS/LAZ内のheader、VLR、EVLRから、WKT、GeoTIFF、EPSG候補、高さ基準の有無を確認するローカル診断です。
座標変換やEPSG DB参照は行いません。

## 取得品質

LAS/LAZの point format、return、scan angle、GPS time などの属性が利用できるかを確認する参考診断です。
測量精度や成果品品質を保証するものではありません。

## 地表候補アシスト

class 2 ground または下位パーセンタイル推定から、表示点ベースの参考地表候補を作る機能です。
測量成果DTM、設計面、切盛根拠ではありません。

## 作業メモコピー

現在の表示、断面、計測、異常候補、取得品質、地表候補の要約JSONを手動でコピーする機能です。
ローカルファイル名、点群本体、元LAS座標配列は含めません。

## WebGPU

ブラウザーでGPUを使うための新しいAPIです。

## WebGL2

ブラウザーで3D描画を行うAPIです。

## Worker

ブラウザー内のバックグラウンドスレッドです。

## WASM

WebAssemblyの略です。C/C++/Rust等で作った高速処理をブラウザーで動かす仕組みです。

## OPFS

Origin Private File Systemの略です。ブラウザー内の高速なローカル保存領域です。

## Client-Max / Server-Zero

サーバーではなく、できる限りユーザーPC側で処理する設計です。
