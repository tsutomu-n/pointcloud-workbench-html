# 用語集

## LAS

点群データの標準的なファイル形式です。

## LAZ

LASを圧縮した形式です。読み込み時に展開処理が必要です。

## 点群

3D空間上の多数の点の集合です。地形、道路、法面、構造物などを表します。

## Preflight

読み込み前の事前確認です。

## 表示率

元点数に対して、実際に表示する点数の割合です。

## WebGPU

ブラウザーでGPUを使うための新しいAPIです。

## WebGL2

ブラウザーで3D描画を行うAPIです。

## Worker

ブラウザー内のバックグラウンドスレッドです。

## WASM

WebAssemblyの略です。C/C++/Rust等で作った高速処理をブラウザーで動かす仕組みです。

## OPFS

Origin Private File Systemの略です。ブラウザー内の高速なローカル保存領域です。

## Client-Max / Server-Zero

サーバーではなく、できる限りユーザーPC側で処理する設計です。
