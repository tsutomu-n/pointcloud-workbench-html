# トラブルシューティング

## アプリが起動しない

確認:

- ChromeまたはEdge最新版か
- ネットワーク接続があるか
- JS/WASMの取得がブロックされていないか
- 社内プロキシやセキュリティソフトでCDNが止められていないか

## LAS/LAZが読み込めない

確認:

- 拡張子が `.las` または `.laz` か
- ファイルサイズ上限を超えていないか
- LASは3GB超で拒否される
- LAZは2GB超で拒否される
- まずLOW品質で試す

## 画面が重い

対応:

1. LOW品質で読み込む。
2. 分類表示ではなく標高表示にする。
3. スライス表示をOFFにする。
4. 他のブラウザータブを閉じる。
5. Chrome/Edgeを再起動する。

## Geo Previewが使えない

確認:

- 点群にCRS情報があるか
- ネットワーク接続があるか
- 地図タイル取得がブロックされていないか
- Hosted High Performance ModeでCOEPにより外部タイルがブロックされていないか

## スクリーンショットに地図が入らない

外部地図タイルのCORS制約により、地図込みスクリーンショットが制限される場合があります。
点群のみのスクリーンショットを使ってください。
