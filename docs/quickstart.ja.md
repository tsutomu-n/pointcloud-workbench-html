# クイックスタート

## 目的

PointCloudWorkbenchを使って、ブラウザー上でLAS/LAZ点群を確認します。

## 必要なもの

- 最新の Google Chrome または Microsoft Edge
- ネットワーク接続
- `.las` または `.laz` ファイル
- 可能ならメモリに余裕のあるPC

Safari、Firefox、古いブラウザーは対応対象外です。

## 使い方

1. `PointCloudWorkbench.html` を開きます。
2. `.las` または `.laz` ファイルを選択します。
3. Preflight画面で以下を確認します。
   - ファイルサイズ
   - 元点数
   - 推定表示点数
   - 表示率
   - メモリリスク
   - 推奨品質
4. `LOW / MEDIUM / HIGH / MAX` から品質を選びます。
5. 読み込みを開始します。
6. 読み込み後、3D/2D、スライス、分類、統計を確認します。
7. 統計パネルで以下を確認します。
   - CRS 診断
   - 取得品質
   - 地表候補
   - 地表方式
8. 状況共有が必要な場合は「診断情報をコピー」または「作業メモコピー」を使います。

## 注意

このアプリは点群ファイルをサーバーへアップロードしません。
選択したLAS/LAZファイルはブラウザー内で処理されます。

ただし、アプリ本体、LAZデコーダ、描画コード、地図タイルなどの取得にはネットワーク接続を使います。

地表候補は表示点ベースの参考診断です。
測量成果DTM、設計面、切盛根拠、納品用地形モデルではありません。
